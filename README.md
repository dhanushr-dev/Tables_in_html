# 📊 HTML Tables Project

A simple and structured **HTML Tables Project** built using **pure HTML5**. This project demonstrates different types of HTML tables, including basic tables, `colspan`, `rowspan`, and a timetable using semantic table elements.

## 🚀 Live Demo

🔗 https://dhanushr-dev.github.io/Tables_in_html/

## 📂 GitHub Repository

🔗 https://github.com/dhanushr-dev/Tables_in_html

---

## 📌 Features

- 📋 Basic HTML Table
- 🔀 Table with `colspan`
- 📑 Table with `rowspan`
- 📅 Timetable Layout
- 🏷️ Table Caption
- 📂 Semantic Table Structure (`thead`, `tbody`, `tfoot`)
- 📏 Custom Border, Cell Padding, and Cell Spacing
- 🎯 Beginner-Friendly HTML Table Examples
